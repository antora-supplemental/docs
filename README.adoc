= Antora Supplemental Docs

Documentation hub for the antora-supplemental organization, built with Antora.
The UI uses the antora-dark-theme release bundle plus a local green brand overlay (`supplemental-ui/`) so this portal is visually distinct from per-project docs on GitHub Pages.

== Build

* Production (fetch from GitHub): `pnpm build`
* Local (sibling repos): `pnpm run build:local`

== Content sources

Docs are pulled from:

* antora-demo-template
* antora-extensions-registry
* antora-themes-site
* dprint-plugin-asciidoc
* asciidoc-interactive (instruction-flow component)

This repo's own content is under `docs/` (portal and nav).

The `package.json` file dependency `file:../asciidoc-interactive` supplies the `@antora-supplemental/instruction-flow` Asciidoctor extension (registered in `antora-playbook.yml`). GitHub Actions clones that repository next to this one before `pnpm install` so the path resolves on CI.
